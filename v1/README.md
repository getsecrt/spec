# v1 Specification Files

This directory is the versioned contract for client/server interoperability.

- `api.md`: HTTP API semantics for v1.
- `envelope.md`: client-side envelope and crypto workflow for v1.
- `server.md`: server runtime behavior for v1.
- `cli.md`: CLI interface and argument semantics for v1 clients.
- `openapi.yaml`: OpenAPI 3.1 schema for v1.
- `envelope.vectors.json`: crypto interop test vectors.
- `cli.vectors.json`: TTL parsing test vectors.
